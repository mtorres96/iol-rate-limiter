prom-client API the executor will use:

  - new Counter({ name, help, labelNames, registers: [register] })
  - counter.inc({ decision: "allowed" | "blocked" })
  - await register.metrics()  // returns Prometheus text
  - register.contentType       // the Prometheus content-type header value
