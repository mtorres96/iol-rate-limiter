Clock — CORE-03 / D-09 (integer ms):
  interface Clock { now(): number; }

OpTuple — D-08 (Store ops return a primitive numeric tuple, NOT a Decision; identical to a Lua EVAL return):
  type OpTuple = [allowed: 0 | 1, remaining: number, resetMs: number, retryAfterMs: number]

Decision — CORE-02 / D-03 / D-04 / D-05 / D-12 (limiter assembles this from the tuple):
  interface Decision { allowed: boolean; limit: number; remaining: number; resetMs: number; retryAfterMs: number; }

RateLimiter — CORE-01 (async; cost optional, default applied in the limiter):
  interface RateLimiter { consume(key: string, cost?: number): Promise<Decision>; }

Store — CORE-04 / D-06 (ONE atomic op per algorithm; each op IS the algorithm; returns OpTuple):
  interface Store {
    tokenBucket(key: string, cfg: TBConfig, cost: number, now: number): OpTuple;
    slidingWindow(key: string, cfg: WindowConfig, cost: number, now: number): OpTuple;
    fixedWindow(key: string, cfg: WindowConfig, cost: number, now: number): OpTuple;
  }

Config shapes — D-10 / D-11:
  interface TBConfig { capacity: number; refillPerInterval: number; intervalMs: number; }
  interface WindowConfig { limit: number; windowMs: number; }
